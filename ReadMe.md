# Print It – Carrousel interactif en JavaScript

Print It est une imprimerie familiale souhaitant dynamiser son site internet statique en y intégrant des éléments interactifs.

Ce projet consiste à ajouter un carrousel fonctionnel en JavaScript afin d’améliorer l’expérience utilisateur et rendre le site plus dynamique.

---

## Objectifs du projet

- Programmer avec les fondamentaux de JavaScript  
- Manipuler le DOM  
- Gérer les événements utilisateurs  
- Intégrer un carrousel interactif  
- Interagir avec une structure HTML existante  

---

## Technologies utilisées

- HTML5  
- CSS3  
- JavaScript  
- Git / GitHub  

---

## Fonctionnalités principales

- Carrousel d’images interactif  
- Navigation via boutons précédent / suivant  
- Mise à jour dynamique du contenu (images, textes)  
- Indicateurs visuels de position (dots)  
- Défilement fluide entre les slides  

---

## JavaScript – Manipulation du DOM

Le projet repose sur l’utilisation de JavaScript pour :

- Sélectionner et modifier des éléments HTML  
- Mettre à jour le contenu dynamiquement  
- Réagir aux interactions utilisateur  
- Gérer les événements (click, navigation)  

---

## Gestion des événements

Plusieurs événements sont utilisés pour rendre le carrousel interactif :

- Click sur les boutons de navigation  
- Click sur les indicateurs (dots)  
- Mise à jour de l’affichage en fonction de l’action utilisateur  

---

## Compétences développées

- Compréhension des fondamentaux JavaScript  
- Manipulation du DOM  
- Gestion des événements utilisateurs  
- Dynamisation d’un site web statique  
- Structuration d’un script JavaScript  
- Utilisation de Git et GitHub pour le versionning
